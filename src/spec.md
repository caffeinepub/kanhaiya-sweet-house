# Specification

## Summary
**Goal:** Create a single-page website for Kanhaiya Sweet House showcasing their Indian sweets business with menu, ordering, and contact information.

**Planned changes:**
- Create header with business name, tagline, and orange gradient background
- Implement navigation menu linking to About, Menu, Order, and Contact sections
- Build About section with business description
- Display menu with three sweet items (Laddu ₹400/kg, Kaju Katli ₹900/kg, Gulab Jamun ₹350/kg) in card layout
- Create order form with name, mobile, and order details inputs
- Add contact section with address and embedded Google Maps
- Apply warm color scheme (#fff3e0 background, orange accents)
- Include footer with copyright notice

**User-visible outcome:** A complete single-page website where visitors can view the sweet menu, place orders through a form, and find the shop's location in Karhal with an embedded map.
